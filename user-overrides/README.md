# User Overrides

Files in this directory are templates for personal configuration.
They are NOT tracked in git once copied to `~/.claude/`.

## Files

- `settings.local.json.template` — Copy to `~/.claude/settings.local.json` for personal settings overrides
- `CLAUDE.md.personal.template` — Add personal identity and preferences to your CLAUDE.md

## Usage

The `install.sh` script automatically copies these templates to `~/.claude/` if they don't already exist.
Edit your local copies — they are gitignored and won't be overwritten on update.
